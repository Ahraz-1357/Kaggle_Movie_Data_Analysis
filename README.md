# IMDb Movies Data Analysis 🎬

## Overview

This project performs **Exploratory Data Analysis (EDA)** on the IMDb Movies dataset to uncover meaningful insights about movie ratings, runtime, genres, voting patterns, and release trends.

Using Python data analysis and visualization libraries, the project explores patterns in movie data through statistical summaries and visual storytelling.

---

## Objectives

The analysis focuses on answering questions such as:

* What are the **top-rated movies**?
* Which movies received the **highest number of votes**?
* What is the distribution of IMDb ratings?
* Are there any **outliers in movie ratings**?
* How does runtime affect movie ratings?
* Which genres perform best on average?
* How have movie ratings changed over time?
